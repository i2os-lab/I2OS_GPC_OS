## Overview

**I2OS_GPC (Infinity Intelligence Operating System – General Purpose Controller)** is a structure-driven market response operating system designed for MetaTrader-based autonomous trading systems.

Unlike conventional Expert Advisors (EAs), I2OS_GPC does not rely on prediction, indicators, or signal-based execution.

Instead, it executes trades only after multi-layer structural verification and real market movement confirmation.
